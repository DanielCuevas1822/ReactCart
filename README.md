# ReactCart

_Simulation of a shopping cart whith React Js using Axios and LocalStorage._

_Concepts learned in this proyect:_

* Axios
* LocalStorage
* React Js

# Start 🚀

_Clone this repo_
```
$ git clone https://github.com/DanielCuevas1822/ReactCart.git
```
Use following commands:
```
$ cd ReactCart
```
```
$ npm install
```
```
$ npm start
```
